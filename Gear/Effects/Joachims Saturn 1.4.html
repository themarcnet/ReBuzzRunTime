Joachim's Saturn filter (14-Jan-07) Planet Filter Series

SATURN - all-purpose vintage degrader

Temperature input
Tone control as the first thing in the input chain

Record noise level
This parameter adds vinyl scratches at 33 rpm. If the value says "Missing file!", you should re-download the plugin from www.buzzmachines.com and make sure that "Joachims Saturn.raw" is located in gear/effects/ (the same directory as Joachims Saturn.dll).

Be careful about replacing this file with another, because your .bmx file will then be incompatible with other people's bmx files. However, I've decided to include some variations anyway. The default is "Joachims Saturn 1.4 06.raw". If you want another kind of vinyl scratch noise, just copy one of the other ones over "Joachims Saturn 1.4.raw".

If you need constant vinyl noise going, there's an attribute for this. The vinyl scratch audio is currently optimized for 96 Khz samplerate. If you work at

Room Size and Room Level
With this you can add reverb to make stuff sound like they're recorded at a distance. This is also a useful trick when trying to make stuff sound retro.
The Room Size parameter also controls the fading out of the vinyl noise - even if Room Level is set to "OFF".

Bit degrader
Maximum bit resolution. This is useful when emulating old digital equipment like e.g. an Amiga that (usually) has only 8-bit sound output.

Samplerate degrader
Also useful when emulating old digital equipment. This filter does not have an anti-aliasing filter, so new frequencies will be introduced.

Antique (with volume compensation)
Makes stuff sound like old rusty equipment by changing the EQ in a special way.

Detune level and Detune speed
Good for simulating record and tape players.

Ring modulation and Ring mod. Mix
Considering the name of this plugin, i just had to add it!   ;-)

Overdrive
Retro gear tend to slightly and gradually overdrive at high gain.

Overdrive Shape
When set to 0 you get transistor-like distortion. When other values might resemble tube distortion more.

Temperature output
Same as Temperature input, but is placed at the output of the effect.

Highpass filter
This 12 dB/oct filter was mainly thought as a DC removal tool, but can ofcourse also be used as a normal highpass filter.
