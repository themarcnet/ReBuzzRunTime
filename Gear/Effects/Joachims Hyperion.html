Joachim's Hyperion filter (7-May-07) Planet Filter Series

HYPERION - 30 band graphical equalizer

Enable
This is a bypass switch to help you compare the original and the new result.

Input gain
To adjust signal strength

Frequency bands
Each band gives you -18 to +18 dB of cut/boost in less than 0.2 dB steps. The bandwidth of each band is controlled using the Bandwidth parameter. When the band is set to 0 dB, it is disabled to prevent extraneous CPU usage. If you use a lot of bands, expect a lot of CPU strain.

Highpass
To remove unwanted subsonics and DC offset, a high quality 12 dB/oct butterworth highpass filter is available.

Bandwidth
This is an unusual feature on a graphical EQ. It allows you to adjust the bandwidth affected by each frequency band. Large bandwidths will cause the bands to overlap and be more smooth sounding. Narrow bands will leave gaps between each frequency band that is not affected by either of the adjacent bands.
